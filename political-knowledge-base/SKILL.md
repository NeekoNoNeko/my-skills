# 政治知识库 — Political Knowledge Base

## 用途

本 Skill 提供《习近平新时代中国特色社会主义思想概论》的**参考文档**：

- **教材**：《习近平新时代中国特色社会主义思想概论》，高等教育出版社、人民出版社，2023年版
- **结构**：导论 + 17 章正文 + 结语
- **语言**：全文中文（关键术语保留英文对照）

适用于：
- 查询习近平新时代中国特色社会主义思想的核心概念和理论
- 查阅教材原文、知识点详解
- 快速定位理论专题所在的章节
- 系统性地复习政治理论知识体系

> 文档采用 MinerU 格式，每个章节为独立目录，内含 `.md` 文档和 `images/` 图片目录，插图引用完整保留。

---

## Agent 行为指南

被触发时按以下流程工作，如果中间已经找到答案不必机械执行全部步骤：

1. **识别专题领域**：判断用户的问题属于哪个专题
   - 基础理论（社会主义、新时代、党的全面领导、以人民为中心）
   - 经济建设（改革开放、高质量发展）
   - 政治建设（全过程人民民主、全面依法治国）
   - 文化建设（文化强国、意识形态）
   - 社会建设（民生保障、社会治理）
   - 生态文明（美丽中国、绿色发展）
   - 安全与国防（国家安全、国防军队）
   - 国家统一与外交（一国两制、大国外交）
   - 党的建设（全面从严治党、反腐败）

2. **查大纲定位**：先读取教材的知识点大纲文件（`XiJinping-Thought-outline.md`），找到对应的章节目录
3. **读取章节文件**：根据大纲定位到具体章节目录，读取其中的 `.md` 文档获取详细内容
4. **交叉引用**：如果概念依赖前置知识（如理解"中国式现代化"需要先理解"中国特色社会主义"），主动回溯到相关章节补充说明
5. **组织回答**：
   - 默认用**中文**回答用户
   - 关键术语保留**原文表述**
   - 如果需要引用文档中的图片，说明图片位置
   - 回答时标注引用出处和参考章节
6. **找不到时**：如果当前知识库没有覆盖用户的问题，如实告知"本知识库未包含该内容"，而不是猜测或编造
